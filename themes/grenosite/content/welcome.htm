Nazywam się Grzegorz Czekaj, jestem programistą aplikacji webowych. Realizuję projekty zarówno front-end jak i back-end. Do budowy stron używam October CMS, systemu zarządzania treścią opartego  o popularny framework Laravel. Jest to system prosty, elastyczny z nowoczesnym wyglądem. Dzięki October CMS klienci oszczędzają zarówno czas jak i pieniądze. Front-end witryn buduję wg techniki RWD (Responsive web design). Na życzenie buduję w oparciu o framework Bootstrap 4.x, lecz jeśli to możliwe w sferze frontendu używam najnowszej technologii jak np. Flexbox czy CSS GRID.

Budowa aplikacji webowej wiąże się z rozbudowanym mechanizmem bazodanowym. W takich przypadkach korzystam z dobrodziejstw frameworka Laravel.
